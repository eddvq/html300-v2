As a transplant to the Pacific Northwest, I had a hard time identifying the local plantlife around me. I figured I wasn't the only one and figured I should build something that could (if I can get it to work) help others.

I'm newish to this coding stuff and reading into new documentation can be a bit intimidating. I figured if I combined something that I love it would make this project less daunting. So far I've learned alot! It's difficult but one step at a time!

It is a huge timesaver. Building this form with custom code would have taken me an eternity. I'm really happy with the outcome, though I had to override some of the styles for the Nav and Footer. I'm thinking of keeping these bootstrap stylings if possible. (just the form and FAQs)
